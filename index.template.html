Extract the zip file content into your 
%Empyrion%\Saves\Cache
folder

  1. Download the zip file: $SHAREDDATAZIPFILENAME
  2. Locate the downloaded zip file in File Explorer.
  3. Right-click on the zip file and select "Extract All..."
  4. As destination, navigate to your Empyrion installation, then select the "Saves" folder, then the "Cache" folder
  5. Make sure zip file's content including all folders ends up in there properly.
  6. You should now have a folder like this:
    %Empyrion%\Saves\Cache\$CACHEFOLDERNAME\SharedData
