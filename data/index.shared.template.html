Extract the zip file content into your 
%Empyrion%\Saves\Cache\$CACHEFOLDERNAME
folder and copy the zip file to 
_SharedDataZips

  1. Download the zip file: $SHAREDDATAAUTOZIPFILE
  2. Create the folder 
    %Empyrion%\Saves\Cache\$CACHEFOLDERNAME
  3. Locate the downloaded zip file in File Explorer.
  4. Right-click on the zip file and select "Extract All..."
  5. As destination, navigate to your Empyrion installation, then select the "Saves" folder, then the "Cache" and the created folder
  6. Make sure zip file's content including all folders ends up in there properly.
  7. You should now have a folder like this:
    %Empyrion%\Saves\Cache\$CACHEFOLDERNAME\SharedData
  8. Copy/move the downloaded file to the folder 
    %Empyrion%\Saves\Cache\_SharedDataZips
  9. Please ask other players or an admin about the filename that that file has to be. If anyone is on the server already, they should check their folder and look for the newest zip file of that server.
  10. Rename the latter to 
    XXXXXXXXXXXXX.zip
    XXXXXXXXXXXXX being the file name you found out in the step above
